# Кликер

Простой кликер с системой достижений

## Описание

Это простой кликер, где вы можете:
* Накапливать клики
* Открывать достижения
* Сохранять прогресс

## Как запустить

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
3. Наслаждайтесь игрой!

## Технологии

* HTML5
* CSS3
* JavaScript
